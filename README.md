# Portfólio Pessoal - José Augusto

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar minha trajetória profissional, competências técnicas e projetos na área de Tecnologia da Informação.

O site destaca minha transição e união entre o suporte de hardware/infraestrutura e o desenvolvimento de software (ADS).

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização moderna e layout responsivo.
- **JavaScript**: Interatividade e efeitos visuais.
- **Font Awesome**: Ícones para representação visual de competências.

## 📋 Seções do Portfólio

- **Home**: Apresentação inicial e link para download do currículo.
- **Sobre**: Resumo da minha trajetória em Juiz de Fora e paixão por tecnologia.
- **Skills**: 
    - **Infraestrutura & TI**: Manutenção de hardware (PC/Gamer/Note) e redes.
    - **Desenvolvimento**: Lógica de programação e Análise de Sistemas.
    - **Complementares**: Soft skills como trabalho em equipe e atendimento.
- **Experiência**: Linha do tempo detalhando passagens pela Cercred, Datafor Informática e AMAC.
- **Contatos**: Links diretos para minhas redes sociais e GitHub.
